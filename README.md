# Beacon - iOS Programming Exam
 
**Course:** iOS Programming (15 ECTS)  
**Institution:** Kristiania University College  
**Tech:** SwiftUI, MapKit, SwiftData, Geoapify Places API
 
Beacon is a place-discovery app built around an interactive map. The case revolves around helping users find nearby restaurants, cafés, and hotels through the Geoapify Places API, view details about each place and save favorites, all stored locally with SwiftData.
